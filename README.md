# osrs-web-scraper
Scrape information from the Oldschool Runescape website.

## Setup

### Install dependencies
```
yarn install
```

### Setup environment
```
NODE_ENV=development
NEWS_LINK=https://secure.runescape.com/m=news/a=1/desert-treasure-ii---the-fallen-empire?oldschool=1
```

### Run the scraper
```
yarn start
```
